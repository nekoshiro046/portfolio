Musical Chair

制作期間 : 進行中プロジェクト

制作人数 : 1名

形式 : オーディオビジュアル・インスタレーション

使用ソフト : Unity / Logic Pro X

XRを用いて名作椅子の素材的,背景的情報を拡張し、新たな音響空間を構築する複数人同時体験型オーディオビジュアル・インスタレーション作品。
現在進行中で制作が行われており、実装環境や機材の最適化を模索している。

About

実世界の名作椅子をXRデバイスを通して認識することで、椅子ごとのサウンドが生成される。
体験者はこの時、それぞれのサウンドの指向性や周波数特性に触れ、椅子の配置をを自らデザインする。椅子の配置,高さによって、サウンドのピッチ,エフェクト等に変化が生まれ多様なオーディオビジュアルが生成される。

Sound

Logic Pro Xを用いて各々の名作椅子のサウンドデザインを行なった。
この12のサウンドは椅子の特性(重さや材質,歴史的背景等)を考慮したものであり、周波数,指向性,非音楽的有無などの差異がある。また紐付けられた名作椅子の高さや向きにより、音のピッチ、方向が変化する。

About the production

学部３年後期からテーマ設定を行い、自身の卒業制作へとつながった作品。
試作段階ではARを用いた実装を行なっていたが、ハード面やUX視点からの難点から現在は実装環境や機材の最適化を模索している。自身が培ってきたメディア領域,サウンド領域,芸術領域の経験を元にXR体験への新たなアプローチを行う。
